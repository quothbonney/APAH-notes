# AP Art History Notes

Free online AP Art History notes. This repository is exported from `Obsidian.md` once a day automatically. The notes are a direct compendium from the Barron's AP Art History Textbook, smarthistory, and lecture notes. 

The notes can be read online on GitHub with the integrated markdown viewer. Just click on a folder and a file and you should be good. If you want a prettier experience, I recommend you download `Obsidian.md` onto your computer. it is free and an awesome technology.
